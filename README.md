# Aprendendo-Python
Repositório criado para acompanhar e registrar o processo de aprendizado da linguagem de programação Python, utilizando como guia o material fornecido por [Fernando Masanori](https://github.com/fmasanori/PPZ) e seu curso [Python para zumbis](https://www.youtube.com/watch?v=YO58tXerKDc&list=PLUukMN0DTKCtbzhbYe2jdF4cr8MOWClXc&index=2).
Nesse repositório compartilharei minha resolução dos exercícios das listas, bem como alguns programas de exemplo que poderei fazer ao longo dos meus estudos.
